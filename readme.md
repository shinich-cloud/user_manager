# User Manager (v3)

Небольшое консольное приложение на Python для управления пользователями.
Проект сделан для практики backend‑разработки: работа с пользователями,
ролями и базой данных.

В первых версиях данные хранились в JSON файле, но в текущей версии
хранение переведено на PostgreSQL.

------------------------------------------------------------------------

## Возможности

-   регистрация пользователей
-   вход в систему
-   хеширование паролей
-   роли пользователей
-   просмотр списка пользователей
-   назначение ролей
-   audit‑log действий

------------------------------------------------------------------------

## Роли

В системе есть три роли:

  Роль        Возможности
  ----------- -----------------------------------
  user        обычный пользователь
  moderator   может просматривать пользователей
  admin       может менять роли пользователей

Первый зарегистрированный пользователь автоматически получает роль
**admin**.

## Запуск проекта

### 1. Установить зависимости

    pip install -r requirements.txt

### 2. Создать базу данных

В PostgreSQL нужно создать базу данных:

    user_manager

### 3. Создать таблицы

    python init_db.py

### 4. Запустить программу

    python main.py
