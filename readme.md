# User Manager

Небольшое консольное backend‑приложение на Python для управления
пользователями. Проект сделан в учебных целях для практики
backend‑разработки и работы с базами данных.

Текущая версия использует PostgreSQL для хранения пользователей.

------------------------------------------------------------------------

## Возможности

-   регистрация пользователей
-   вход в систему
-   хеширование паролей
-   роли пользователей
-   просмотр пользователей
-   назначение ролей
-   audit‑log действий

------------------------------------------------------------------------

## Роли пользователей

  Роль        Описание
  ----------- -----------------------------------
  user        обычный пользователь
  moderator   может просматривать пользователей
  admin       может менять роли пользователей

Первый зарегистрированный пользователь автоматически получает роль
**admin**.

------------------------------------------------------------------------

## Технологии

-   Python
-   PostgreSQL

------------------------------------------------------------------------

## Быстрый запуск

### 1. Установить зависимости

    pip install -r requirements.txt

### 2. Создать базу данных

Создать базу данных в PostgreSQL:

    user_manager

### 3. Создать таблицы

    python init_db.py

### 4. Запустить приложение

    python main.py
