Консольный backend-проект на Python для управления пользователями с ролями.

Проект развивается поэтапно: от простого CLI до полноценного backend-сервиса.

---

## 🆕 Обновление
### Версия 2.0
- Добавлена роль admin
- Первый зарегистрированный пользователь автоматически становится admin
- Реализовано разграничение прав доступа
---

## 🚀 Функциональность
- Регистрация пользователей
- Авторизация (логин)
- Хеширование паролей (SHA-256)
- Роли пользователей:
  - user — базовый доступ
  - admin — расширенные права
- Хранение данных в JSON
---

## 👑 Возможности admin
- Просмотр списка всех пользователей
- Назначение и изменение ролей (`user` / `admin`)
- Контроль доступа к функциональности
