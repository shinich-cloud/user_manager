# User Manager v2.0.0

User Manager — консольное приложение на Python для управления пользователями
с ролевой моделью доступа и аудитом действий.

Проект создан в учебных целях для практики:
- работы с файлами
- проектирования ролей и прав доступа
- структурирования backend-кода

---

## 🚀 Возможности

- Регистрация и авторизация пользователей
- Хеширование паролей (SHA-256)
- Ролевая модель доступа:
  - **user** — базовый пользователь
  - **moderator** — просмотр пользователей
  - **admin** — управление ролями
- Audit-log всех ключевых действий
- Разделение логики по модулям

---

## 🧑‍💻 Роли и права доступа

| Роль       | Вход | Просмотр пользователей | Назначение ролей |
|-----------|------|------------------------|------------------|
| user      | ✅   | ❌                     | ❌               |
| moderator | ✅   | ✅                     | ❌               |
| admin     | ✅   | ✅                     | ✅               |

📌 Первый зарегистрированный пользователь автоматически получает роль **admin**.

## 📝 Audit-log

Все ключевые действия записываются в `audit_logs.json`.

Пример записи:
json
{
  "time": "2026-02-09 18:30:21",
  "user": "admin",
  "action": "change_role",
  "target": "username"
}
